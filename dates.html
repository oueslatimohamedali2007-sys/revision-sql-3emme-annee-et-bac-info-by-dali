Chapitre 7

Fonctions Temporelles

Extraire ou manipuler des formats temporels pour analyser des événements datés.

NOW() — Date et Heure Actuelle

Retourne la date et l'heure du système au moment de l'exécution.

-- Récupérer la date et heure actuelles
SELECT NOW();
-- Résultat: 2026-05-19 14:30:45

-- Insérer un enregistrement avec la date courante
INSERT INTO logs (message, timestamp) VALUES ('Connexion', NOW());

DATE() — Extraire la Date

Extrait uniquement la date d'une valeur DATETIME (sans l'heure).

-- Extraire la date d'un timestamp
SELECT DATE(NOW());
-- Résultat: 2026-05-19

-- Comparer des dates
SELECT * FROM eleves WHERE DATE(date_inscription) = '2026-05-19';

DAY(), MONTH(), YEAR() — Extraire des Composants

Extrait le jour, mois ou année d'une date.

-- Extraire le jour (1-31)
SELECT DAY(date_naissance) FROM eleves;

-- Extraire le mois (1-12)
SELECT MONTH(date_inscription) FROM eleves;

-- Extraire l'année
SELECT YEAR(date_naissance) FROM eleves;

-- Exemple combiné: élèves nés en juin
SELECT * FROM eleves WHERE MONTH(date_naissance) = 6;

DATEDIFF() — Différence Entre Deux Dates

Calcule le nombre de jours entre deux dates.

-- Différence en jours
SELECT DATEDIFF('2026-06-01', '2026-05-19') AS jours_restants;
-- Résultat: 13

-- Âge approximatif en jours
SELECT nom, DATEDIFF(NOW(), date_naissance) AS jours_vecus 
FROM eleves;

-- Temps depuis l'inscription
SELECT nom, DATEDIFF(NOW(), date_inscription) AS jours_inscrits 
FROM eleves 
ORDER BY jours_inscrits DESC;

DATE_ADD() et DATE_SUB() — Ajouter/Soustraire des Jours

Ajoute ou soustrait un intervalle de temps à une date.

-- Ajouter 30 jours
SELECT DATE_ADD(NOW(), INTERVAL 30 DAY) AS dans_30j;

-- Soustraire 1 mois
SELECT DATE_SUB(NOW(), INTERVAL 1 MONTH) AS il_y_a_un_mois;

-- Date limite (fin de mois)
SELECT DATE_ADD(NOW(), INTERVAL 5 DAY) AS date_limite;

💡 Cas d'usage pratiques

Applications réelles des fonctions temporelles:

-- Inscrits aujourd'hui
SELECT COUNT(*) FROM eleves WHERE DATE(date_inscription) = DATE(NOW());

-- Anniversaires ce mois
SELECT nom, MONTH(date_naissance) FROM eleves 
WHERE MONTH(date_naissance) = MONTH(NOW());

-- Âge approximatif en années
SELECT nom, YEAR(NOW()) - YEAR(date_naissance) AS age_approx 
FROM eleves;

-- Rapport par mois d'inscription
SELECT 
  MONTH(date_inscription) AS mois, 
  YEAR(date_inscription) AS annee,
  COUNT(*) AS inscriptions 
FROM eleves 
GROUP BY YEAR(date_inscription), MONTH(date_inscription)
ORDER BY annee DESC, mois DESC;
