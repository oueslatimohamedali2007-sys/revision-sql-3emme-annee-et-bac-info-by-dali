Chapitre 8

Calculs Statistiques (Agrégations)

Calculs globaux appliqués sur un ensemble de lignes pour résumer les données.

COUNT() — Comptage de lignes

Compte le nombre de lignes ou de valeurs non NULL dans une colonne.

-- Compter toutes les lignes
SELECT COUNT(*) AS total_eleves FROM eleves;

-- Compter les valeurs non NULL
SELECT COUNT(email) AS eleves_avec_email FROM eleves;

-- Compter les valeurs distinctes
SELECT COUNT(DISTINCT classe) AS nombre_classes FROM eleves;

SUM() — Somme de valeurs

Calcule la somme totale des valeurs numériques dans une colonne.

-- Somme totale des notes
SELECT SUM(note) AS total_points FROM eleves;

-- Somme avec condition
SELECT SUM(note) AS somme FROM eleves WHERE classe = '3STI';

-- Somme par groupe
SELECT classe, SUM(note) AS total_classe 
FROM eleves 
GROUP BY classe;

AVG() — Moyenne de valeurs

Calcule la moyenne (moyenne arithmétique) des valeurs numériques.

-- Moyenne générale
SELECT AVG(note) AS moyenne_classe FROM eleves;

-- Moyenne par classe
SELECT classe, AVG(note) AS moyenne 
FROM eleves 
GROUP BY classe;

-- Arrondir la moyenne
SELECT ROUND(AVG(note), 2) AS moyenne_arrondie FROM eleves;

MAX() et MIN() — Valeurs extrêmes

Trouvent la valeur maximale et minimale dans une colonne.

-- Meilleure et pire note
SELECT MAX(note) AS meilleure_note, MIN(note) AS pire_note FROM eleves;

-- Avec GROUP BY
SELECT classe, MAX(note) AS meilleure, MIN(note) AS pire 
FROM eleves 
GROUP BY classe;

-- Plus vieux et plus jeune élève
SELECT MAX(date_naissance) AS plus_jeune, MIN(date_naissance) AS plus_vieux 
FROM eleves;

GROUP BY — Grouper par critères

Regroupe les lignes par une ou plusieurs colonnes avant d'appliquer les agrégations.

-- Notes par classe
SELECT classe, COUNT(*) AS nb_eleves, AVG(note) AS moyenne 
FROM eleves 
GROUP BY classe;

-- Regrouper par plusieurs colonnes
SELECT classe, genre, COUNT(*) AS nombre 
FROM eleves 
GROUP BY classe, genre;

-- Grouper avec ORDER BY
SELECT classe, SUM(note) AS total 
FROM eleves 
GROUP BY classe 
ORDER BY total DESC;

HAVING — Filtrer après agrégation

Applique des conditions sur les résultats agrégés (similaire à WHERE mais après GROUP BY).

-- Classes avec moyenne supérieure à 14
SELECT classe, AVG(note) AS moyenne 
FROM eleves 
GROUP BY classe 
HAVING AVG(note) > 14;

-- Classes avec plus de 20 élèves
SELECT classe, COUNT(*) AS nombre 
FROM eleves 
GROUP BY classe 
HAVING COUNT(*) > 20;

-- Combiner WHERE et HAVING
SELECT classe, AVG(note) AS moyenne, COUNT(*) AS nombre
FROM eleves 
WHERE annee = 2026 
GROUP BY classe 
HAVING COUNT(*) >= 15;

💡 Cas d'usage pratiques

Exemples complets montrant les agrégations en contexte réel:

-- Tableau de bord école
SELECT 
  classe,
  COUNT(*) AS total_eleves,
  AVG(note) AS moyenne_classe,
  MAX(note) AS meilleure_note,
  MIN(note) AS pire_note,
  COUNT(CASE WHEN note >= 10 THEN 1 END) AS reussis
FROM eleves 
GROUP BY classe 
HAVING COUNT(*) > 0
ORDER BY moyenne_classe DESC;

-- Analyse par période
SELECT 
  MONTH(date_inscription) AS mois,
  COUNT(*) AS nouvelles_inscriptions
FROM eleves 
WHERE YEAR(date_inscription) = 2026
GROUP BY MONTH(date_inscription)
ORDER BY mois;
