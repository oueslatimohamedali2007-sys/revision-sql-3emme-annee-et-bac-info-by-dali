Consultation et Affichage (DQL)

Interroger et filtrer les informations de la base de données avec précision.

SELECT — La Base de la Consultation

Sélectionne et affiche les colonnes spécifiées d'une table.

-- Toutes les colonnes
SELECT * FROM eleves;

-- Colonnes spécifiques
SELECT nom, prenom, note FROM eleves;

-- Alias de colonne (renommer l'affichage)
SELECT nom AS 'Nom Complet', note AS 'Note Finale' FROM eleves;

DISTINCT — Éliminer les Doublons

Affiche uniquement les valeurs uniques, éliminant les répétitions.

-- Classes uniques
SELECT DISTINCT classe FROM eleves;

-- Combinaison de colonnes distinctes
SELECT DISTINCT classe, genre FROM eleves;

-- Nombre de classes différentes
SELECT COUNT(DISTINCT classe) FROM eleves;

WHERE — Filtrer les Résultats

Restreint les résultats selon des conditions spécifiques.

-- Condition simple
SELECT * FROM eleves WHERE classe = '3STI';

-- Condition numérique
SELECT * FROM eleves WHERE note > 15;

-- Condition combinée
SELECT * FROM eleves WHERE classe = '3STI' AND note >= 12;

ORDER BY — Trier les Résultats

Organise les résultats par ordre croissant (ASC) ou décroissant (DESC).

-- Ordre alphabétique des noms
SELECT * FROM eleves ORDER BY nom ASC;

-- Notes du plus haut au plus bas
SELECT * FROM eleves ORDER BY note DESC;

-- Tri multiple: classe puis note
SELECT * FROM eleves ORDER BY classe ASC, note DESC;

-- Tri par ordre numérique de colonne
SELECT nom, note FROM eleves ORDER BY 2 DESC; -- Tri par 2e colonne

LIMIT — Limiter le Nombre de Résultats

Restrein le nombre de lignes retournées.

-- Premiers 10 élèves
SELECT * FROM eleves LIMIT 10;

-- Résultats 11 à 20 (offset 10, limit 10)
SELECT * FROM eleves LIMIT 10 OFFSET 10;

-- Top 5 des meilleures notes
SELECT * FROM eleves ORDER BY note DESC LIMIT 5;

GROUP BY — Regrouper les Données

Regroupe les lignes par des valeurs communes, souvent utilisé avec des agrégations.

-- Nombre d'élèves par classe
SELECT classe, COUNT(*) AS nombre FROM eleves GROUP BY classe;

-- Moyenne par classe
SELECT classe, AVG(note) AS moyenne FROM eleves GROUP BY classe;

-- Grouper par plusieurs colonnes
SELECT classe, genre, COUNT(*) AS nombre FROM eleves GROUP BY classe, genre;

HAVING — Filtrer Après Regroupement

Applique des conditions sur les résultats agrégés (WHERE filtre avant GROUP BY, HAVING après).

-- Classes avec plus de 20 élèves
SELECT classe, COUNT(*) AS nombre FROM eleves GROUP BY classe HAVING COUNT(*) > 20;

-- Moyennes supérieures à 14
SELECT classe, AVG(note) AS moyenne FROM eleves GROUP BY classe HAVING AVG(note) > 14;

-- Combiner WHERE et HAVING
SELECT classe, COUNT(*) AS nombre, AVG(note) AS moyenne 
FROM eleves 
WHERE annee = 2026 
GROUP BY classe 
HAVING COUNT(*) >= 15;

💡 Requête Complète Combinée

Exemple complet mêlant tous les éléments:

-- Obtenir les classes avec les meilleures moyennes
SELECT 
  classe,
  COUNT(*) AS nombre_eleves,
  AVG(note) AS moyenne_classe,
  MAX(note) AS meilleure_note
FROM eleves 
WHERE annee = 2026 
GROUP BY classe 
HAVING COUNT(*) >= 10
ORDER BY moyenne_classe DESC
LIMIT 5;
