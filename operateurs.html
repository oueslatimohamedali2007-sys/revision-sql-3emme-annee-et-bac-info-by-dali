Chapitre 6

Opérateurs Logiques et de Comparaison

Créer des filtres conditionnels précis dans vos clauses WHERE.

Opérateurs de Comparaison Basiques

Comparer les valeurs dans les conditions WHERE.

-- Égalité
SELECT * FROM eleves WHERE classe = '3STI';

-- Inégalité (deux syntaxes équivalentes)
SELECT * FROM eleves WHERE note != 15;
SELECT * FROM eleves WHERE note <> 15;

-- Supérieur / Inférieur
SELECT * FROM eleves WHERE note > 12;
SELECT * FROM eleves WHERE note <= 16;

-- Combinaisons
SELECT * FROM eleves WHERE note >= 10 AND note <= 20;

LIKE — Recherche par Motif

Recherche des chaînes de caractères correspondant à un motif. % = zéro ou plusieurs caractères, _ = exactement un caractère.

-- Noms commençant par 'A'
SELECT * FROM eleves WHERE nom LIKE 'A%';

-- Noms se terminant par 'i'
SELECT * FROM eleves WHERE nom LIKE '%i';

-- Noms contenant 'ali'
SELECT * FROM eleves WHERE nom LIKE '%ali%';

-- Noms de 4 caractères commençant par 'B'
SELECT * FROM eleves WHERE nom LIKE 'B___';

BETWEEN — Intervalle de Valeurs

Sélectionne les valeurs dans une plage (inclusive des bornes).

-- Notes entre 10 et 16 (inclus)
SELECT * FROM eleves WHERE note BETWEEN 10 AND 16;

-- Dates dans une période
SELECT * FROM eleves WHERE date_inscription BETWEEN '2026-01-01' AND '2026-05-19';

-- Équivalent avec AND
SELECT * FROM eleves WHERE note >= 10 AND note <= 16;

IN — Appartenance à une Liste

Vérifie si une valeur est dans une liste prédéfinie.

-- Élèves de certaines classes
SELECT * FROM eleves WHERE classe IN ('3STI', '4STI', 'ISI');

-- Certains identifiants
SELECT * FROM eleves WHERE id IN (1, 5, 10, 23);

-- Équivalent avec OR
SELECT * FROM eleves WHERE classe = '3STI' OR classe = '4STI' OR classe = 'ISI';

IS NULL / IS NOT NULL — Valeurs Nulles

Teste la présence ou l'absence de valeur (NULL = pas de valeur).

-- Élèves sans email enregistré
SELECT * FROM eleves WHERE email IS NULL;

-- Élèves avec date de naissance
SELECT * FROM eleves WHERE date_naissance IS NOT NULL;

-- Note manquante
SELECT * FROM eleves WHERE note IS NULL;

AND, OR, NOT — Opérateurs Logiques

Combine plusieurs conditions pour créer des filtres complexes.

-- AND: toutes les conditions doivent être vraies
SELECT * FROM eleves WHERE classe = '3STI' AND note > 15;

-- OR: au moins une condition doit être vraie
SELECT * FROM eleves WHERE classe = '3STI' OR classe = '4STI';

-- NOT: inverse une condition
SELECT * FROM eleves WHERE NOT (classe = '3STI');

-- Combinaison complexe
SELECT * FROM eleves 
WHERE (classe = '3STI' OR classe = '4STI') 
  AND note >= 12 
  AND email IS NOT NULL;

💡 Cas d'usage pratiques

Exemples combinant plusieurs opérateurs:

-- Recherche avancée: élèves de 3STI avec bonne note
SELECT * FROM eleves 
WHERE classe = '3STI' 
  AND note > 14 
  AND date_inscription BETWEEN '2025-09-01' AND '2026-05-31';

-- Filtre complexe: noms commençant par A-C ou D
SELECT * FROM eleves 
WHERE (nom LIKE 'A%' OR nom LIKE 'B%' OR nom LIKE 'C%' OR nom LIKE 'D%')
  AND note IS NOT NULL;

-- Exclusion: tout sauf les 3STI avec note faible
SELECT * FROM eleves 
WHERE NOT (classe = '3STI' AND note < 10);
